Claude Rules 설치 가이드

Claude Code 공통 규칙 배포 매뉴얼 (Windows)

이 매뉴얼은 무엇인가요?
D-ONWORKS 팀에서 사용하는 Claude Code 공통 규칙(CLAUDE.md)과 설정(managed-settings.json)을 새 PC에 설치하거나 업데이트하는 방법을 안내합니다. 총 2단계로, 5분이면 완료됩니다.
1
규칙 파일 다운로드 (Git Clone)
GitHub에서 규칙 파일을 내 PC로 복사합니다. 처음 1번만 하면 됩니다.
작업표시줄 검색에서 PowerShell 또는 명령 프롬프트(cmd)를 검색하여 엽니다.
아래 명령어를 복사 → 붙여넣기 → Enter 합니다.
> git clone https://github.com/d-onworks/claude-rules.git C:\src\claude-rules
이미 한번 실행한 적 있다면 이 단계는 건너뛰세요. 2단계에서 자동으로 최신 버전을 받습니다.
2
규칙 설치 (관리자 권한 필요)
다운로드한 규칙 파일을 Claude Code가 읽는 위치에 복사합니다.
C:\Program Files 폴더에 쓰기 때문에 관리자 권한이 필요합니다.
작업표시줄에서 PowerShell을 검색합니다.
"관리자 권한으로 실행"을 클릭합니다. (우클릭 또는 오른쪽 메뉴)
아래 명령어를 복사 → 붙여넣기 → Enter 합니다.
PS(관리자)> powershell -ExecutionPolicy Bypass -File C:\src\claude-rules\deploy.ps1
"관리자 권한으로 실행"이 안 보이면, PowerShell 아이콘을 우클릭하면 나옵니다.
✓
설치 완료 확인
아래와 같은 메시지가 나오면 성공입니다.
설치 완료!
- 규칙 위치: C:\Program Files\ClaudeCode\CLAUDE.md
- 설정 위치: C:\Program Files\ClaudeCode\managed-settings.json
- Claude Code 재시작 시 자동 적용됩니다.
